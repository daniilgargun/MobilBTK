# Мини-игра "Блок-пазл"

## Структура директории

Мини-игра структурирована следующим образом:

```
lib/minigame/
├── index.dart              # Главный файл для экспорта всех компонентов
├── models/                 # Модели данных
│   ├── cell.dart           # Модель клетки игрового поля
│   ├── block.dart          # Модель блока
│   ├── profession_type.dart # Перечисление типов профессий
│   ├── game_state.dart     # Перечисление состояний игры
│   ├── background_particle.dart # Модель частицы фона
│   └── models.dart         # Файл для экспорта всех моделей
├── painters/               # Классы для отрисовки
│   ├── feedback_block_painter.dart # Отрисовка блока при перетаскивании
│   ├── background_painter.dart # Отрисовка анимированного фона
│   ├── block_preview_painter.dart # Отрисовка превью расположения блока
│   └── painters.dart       # Файл для экспорта всех painters
├── services/               # Сервисы
│   ├── game_service.dart   # Сервис для работы с игровыми данными
│   └── services.dart       # Файл для экспорта всех сервисов
├── widgets/                # Виджеты
│   ├── animated_background.dart # Виджет анимированного фона
│   ├── points_animation.dart # Виджет анимации начисления очков
│   ├── combo_points_display.dart # Виджет для отображения комбо
│   └── widgets.dart        # Файл для экспорта всех виджетов
└── screens/                # Экраны
    └── minigame_screen.dart # Основной экран игры
```

## Описание игры

"Блок-пазл" - это головоломка, в которой игрок размещает блоки различной формы на игровой доске 10x10. Цель игры - очистить как можно больше строк и столбцов, размещая блоки таким образом, чтобы они заполняли полные строки или столбцы.

### Правила игры:
1. Доступно три блока различной формы.
2. Игрок может перетаскивать блоки на игровое поле.
3. Когда блок помещается на поле, он заменяется новым случайным блоком.
4. Когда строка или столбец полностью заполнены, они исчезают и игрок получает очки.
5. Если заполнено несколько строк или столбцов одновременно, запускается комбо, умножающее очки.
6. Игра заканчивается, когда ни один из доступных блоков не может быть размещен на поле.

## Особенности

- Красивый анимированный фон
- Анимации подсчета очков и комбо
- Темная и светлая темы
- Сохранение состояния игры и рекордов
- Адаптивный размер игрового поля
- Множество разных форм блоков для разнообразия
