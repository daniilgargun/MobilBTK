# Журнал изменений

## Версия 1.0.8 (2025-10--07)
* ✨ **Новый раздел персонализации интерфейса**
  * Объединены настройки темы и интерфейса в один раздел
  * Добавлен выбор цветовой темы с визуальным предпросмотром
  * Настройка размера текста теперь работает корректно во всём приложении
  * Выбор формата отображения расписания (список/сетка) с предпросмотром
* 🎨 **Улучшения дизайна**
  * Полностью переработан экран настроек интерфейса с современным дизайном
  * Улучшены карточки занятий в режиме сетки - более компактные и аккуратные
  * Добавлен отступ от цветной линии в карточках для лучшей читаемости
  * Оптимизированы размеры элементов для лучшего использования пространства экрана
* 🔧 **Исправления и оптимизации**
  * Исправлена прозрачность кнопок "Обновить" и "Поделиться" - теперь всегда прозрачные
  * Исправлено применение размера текста при изменении настроек
  * Убраны визуальные "прыжки" при выборе элементов в настройках
  * Улучшена компактность карточек в режиме сетки без потери читаемости
  * В режиме списка подгруппа отображается полностью, в сетке - сокращённо ("Пг")
