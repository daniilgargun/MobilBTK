# Предложения по улучшению приложения "БТК Расписание"

## 🎯 Приоритетные функции (высокий приоритет)

### 1. 🔔 Уведомления о занятиях
- **Описание:** Push-уведомления за 15-30 минут до начала занятия
- **Преимущества:** 
  - Студенты не пропустят занятия
  - Можно настроить расписание уведомлений
  - Работает в фоновом режиме
- **Технические детали:**
  - Использовать `flutter_local_notifications`
  - Планирование уведомлений через `timezone` пакет
  - Настройки: время напоминания, дни недели, отключение для выходных

### 2. 📊 Статистика посещаемости
- **Описание:** Отслеживание посещенных занятий с возможностью добавления отметок
- **Преимущества:**
  - Студенты могут отслеживать свою посещаемость
  - Визуализация статистики (графики, проценты)
  - Экспорт данных
- **Технические детали:**
  - Новая таблица в БД для отметок посещаемости
  - Виджеты для отображения статистики
  - Фильтры по предметам, преподавателям, периодам

### 3. 🔍 Расширенный поиск и фильтры
- **Описание:** 
  - Поиск по времени занятий
  - Фильтр по типам занятий (лекция, практика, лабораторная)
  - Поиск по преподавателям с автодополнением
  - Сохранение часто используемых фильтров
- **Преимущества:** Быстрый доступ к нужной информации
- **Технические детали:**
  - Улучшение существующего поиска
  - Кэширование популярных запросов
  - История поиска

### 4. 📅 Экспорт расписания
- **Описание:** 
  - Экспорт в календарь устройства (iCal формат)
  - Экспорт в PDF
  - Отправка расписания по email
- **Преимущества:**
  - Интеграция с системным календарем
  - Возможность печати расписания
  - Удобный обмен расписанием
- **Технические детали:**
  - Использовать `icalendar` пакет для iCal
  - `pdf` пакет для генерации PDF
  - `mailer` для отправки email

### 5. 🎨 Персонализация интерфейса
- **Описание:**
  - Выбор цветовой схемы приложения
  - Настройка размера шрифта
  - Выбор формата отображения расписания (список/сетка)
  - Темы оформления (не только светлая/темная)
- **Преимущества:** Улучшение UX, доступность
- **Технические детали:**
  - Расширение настроек темы
  - Сохранение пользовательских предпочтений

## 🚀 Средний приоритет

### 6. 📝 Расширенные заметки
- **Описание:**
  - Прикрепление файлов к заметкам
  - Голосовые заметки
  - Напоминания о заметках
  - Поиск по заметкам
- **Преимущества:** Более функциональная система заметок
- **Технические детали:**
  - `file_picker` для выбора файлов
  - `record` для записи аудио
  - Индексация заметок для быстрого поиска

### 7. 👥 Социальные функции
- **Описание:**
  - Обмен расписанием с друзьями
  - Комментарии к занятиям (анонимные)
  - Рейтинг преподавателей (опционально)
- **Преимущества:** Социальная составляющая приложения
- **Технические детали:**
  - Backend для хранения комментариев (Firebase или собственный)
  - Система модерации контента

### 8. 📈 Аналитика и отчеты
- **Описание:**
  - Статистика по нагрузке (количество пар в день/неделю)
  - Анализ свободного времени
  - Отчеты по предметам
- **Преимущества:** Планирование времени, анализ учебной нагрузки
- **Технические детали:**
  - Виджеты графиков (использовать `fl_chart`)
  - Расчет статистики на основе данных расписания

### 9. 🌐 Мультиязычность
- **Описание:** Поддержка белорусского и английского языков
- **Преимущества:** Расширение аудитории
- **Технические детали:**
  - Использовать `flutter_localizations`
  - Локализация всех строк приложения
  - Автоматическое определение языка системы

### 10. 🔄 Автоматическое обновление расписания
- **Описание:**
  - Фоновая синхронизация по расписанию
  - Уведомления об изменениях в расписании
  - История изменений
- **Преимущества:** Всегда актуальное расписание
- **Технические детали:**
  - `workmanager` для фоновых задач
  - Сравнение версий расписания
  - Логирование изменений
