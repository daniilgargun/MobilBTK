# БТК Расписание - Мобильное приложение

**Copyright (c) 2024 Daniil Gargun. All rights reserved.**

## Описание

Мобильное приложение для просмотра расписания занятий Белорусского торгово-экономического колледжа потребительской кооперации. Приложение разработано с использованием Flutter и предоставляет удобный интерфейс для студентов и преподавателей.

Текущая версия приложения: **1.0.11 (build 15)**.

## Основные возможности

- 📅 Просмотр расписания занятий
- 🔍 Поиск по группам, преподавателям, предметам и кабинетам
- 📝 Добавление заметок к датам
- 🌙 Темная и светлая тема и расширенные настройки персонализации интерфейса (цветовая тема, размер текста, режим отображения расписания: список/сетка)
- 📱 Офлайн режим работы
- 📤 Функция "Поделиться расписанием"
- 🔔 Просмотр и напоминания расписания звонков
- 📆 Уведомления об обновлении расписания (при поддержке сервера)
- 🧩 Виджеты расписания и расписания звонков для главного экрана (Android)


## Технические характеристики

- **Платформа:** Flutter (Android/iOS)
- **Язык программирования:** Dart
- **База данных:** SQLite
- **Архитектура:** Provider + Clean Architecture
- **Минимальная версия Android:** 8.0 (API 26)
